# TOOLKIT_SPEC

## Категории

- JSON Tools: formatter, validator, diff, schema validator, sample generator.
- API Tools: curl builder, endpoint checklist, status helper, error model designer.
- Requirements Tools: requirement generator, ambiguity detector, user story, use case, acceptance criteria, traceability matrix.
- SQL Tools: cheat sheet, JOIN helper, aggregates, window functions, typical analyst queries.
- Integration Tools: scenario template, sync/async helper, webhook checklist, retry policy, mapping table.
- NFR Tools: performance, reliability, security, observability, availability calculator.
- Decision Tools: ADR generator, decision log, trade-off matrix, risk matrix.

## Требование

Каждый инструмент работает локально, имеет русские подписи и выдаёт применимый рабочий результат.
