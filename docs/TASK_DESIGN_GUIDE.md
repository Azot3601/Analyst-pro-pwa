# TASK_DESIGN_GUIDE

## Структура задачи

Каждая задача содержит:

- бизнес-контекст;
- зачем это аналитику;
- исходные артефакты;
- что нужно сделать;
- критерий правильности;
- многоуровневые подсказки;
- объяснение;
- частые ошибки;
- связанные знания.

## Подсказки

1. Лёгкий намёк.
2. Направление решения.
3. Структурная подсказка.
4. Почти готовое решение.
5. Разбор после ответа.
