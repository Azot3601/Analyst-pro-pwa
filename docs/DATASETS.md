# DATASETS

## Используемые локальные синтетические датасеты

### Ecommerce

- Ссылка: локальный seed.
- Лицензия: создано для проекта.
- Назначение: заказы, платежи, доставка, статусы, клиенты без персональных данных.
- Задачи: SQL JOIN, поиск расхождений, API history, интеграция доставки.
- Ограничения: синтетический объём небольшой.
- Статус: используется.

### Banking-lite

- Ссылка: локальный seed.
- Лицензия: создано для проекта.
- Назначение: счета, операции, лимиты, статусы без ПДн.
- Задачи: идемпотентность, reconciliation, audit trail.
- Ограничения: не имитирует реальные банковские регламенты.
- Статус: используется.

### Logistics

- Ссылка: локальный seed.
- Лицензия: создано для проекта.
- Назначение: отправления, маршруты, события доставки.
- Задачи: webhooks, async events, SLA.
- Ограничения: упрощённая маршрутизация.
- Статус: используется.

### Education Platform

- Ссылка: локальный seed.
- Лицензия: создано для проекта.
- Назначение: курсы, уроки, прогресс, воронки.
- Задачи: cohorts-lite, требования, отчётность.
- Ограничения: нет персональных данных.
- Статус: используется.

### Support Tickets

- Ссылка: локальный seed.
- Лицензия: создано для проекта.
- Назначение: обращения, статусы, SLA, каналы.
- Задачи: НФТ, мониторинг, приоритизация.
- Ограничения: синтетические тексты.
- Статус: используется.

### API Logs

- Ссылка: локальный seed.
- Лицензия: создано для проекта.
- Назначение: запросы API, статусы, latency, trace id.
- Задачи: анализ ошибок, мониторинг, rate limits.
- Ограничения: сэмпл для обучения.
- Статус: используется.

## Оценённые внешние источники

### Chinook Database

- Ссылка: https://github.com/lerocha/chinook-database
- Лицензия: репозиторий содержит `LICENSE.md`; перед копированием данных требуется финальная проверка текста лицензии.
- Назначение: digital media store.
- Задачи: SQL JOIN, агрегаты, отчётность.
- Ограничения: домен менее близок к системному анализу интеграций.
- Статус: оценён, не добавлен в репозиторий.

### Sakila Sample Database

- Ссылка: https://dev.mysql.com/doc/sakila/en/
- Лицензия: есть отдельный раздел License for the Sakila Sample Database в документации MySQL.
- Назначение: DVD rental domain.
- Задачи: SQL, ERD, нормализация.
- Ограничения: лицензия и перенос в проект требуют отдельной проверки.
- Статус: оценён, не добавлен в репозиторий.

### Microsoft SQL Server Samples / AdventureWorks

- Ссылка: https://github.com/microsoft/sql-server-samples
- Лицензия: MIT для samples/templates по корневому `license.txt`.
- Назначение: OLTP/DW sample databases.
- Задачи: сложное моделирование и отчётность.
- Ограничения: тяжёлый домен, SQL Server-ориентированность.
- Статус: оценён, не добавлен в репозиторий.
