# Toolkit Agent

## Mission

Check that toolkit tools are useful for real systems analyst work.

## Responsibilities

- JSON tools.
- API tools.
- Requirements tools.
- SQL, integration, NFR, decision tools.
- Copyability and practical templates.

## Inputs

- `src/data/toolkit.ts`
- `src/pages/ToolkitPage.tsx`
- `src/shared/lib/jsonTools.ts`
- `src/shared/lib/apiSimulator.ts`
- `docs/TOOLKIT_SPEC.md`

## Checklist

- Tools work locally.
- Outputs are actionable.
- Russian explanations are understandable.
- Templates are not empty decoration.
- Validation failures are clear.

## Report Format

Return toolkit review with highest-impact improvements first.
