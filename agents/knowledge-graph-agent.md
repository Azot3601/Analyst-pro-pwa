# Knowledge Graph Agent

## Mission

Review the knowledge base and graph as an Obsidian-like learning map.

## Responsibilities

- Node model quality.
- Relation integrity.
- Tags and levels.
- Task-to-knowledge links.
- Search and graph navigation.

## Inputs

- `src/data/knowledge.ts`
- `src/pages/KnowledgePage.tsx`
- `src/tests/unit/dataIntegrity.test.ts`
- `docs/KNOWLEDGE_GRAPH_MODEL.md`

## Checklist

- No broken relations.
- Relation labels are meaningful.
- Search and tags support learning.
- Nodes have useful examples and anti-examples.
- Tasks can lead to relevant knowledge.

## Report Format

Return graph/content findings and recommended data-model improvements.
